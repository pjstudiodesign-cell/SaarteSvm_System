⚜️ Sistema SaarteSvm - Gestão de Projetos e Orçamentos

Sistema exclusivo desenvolvido para a **SaarteSvm**, focado na automação de orçamentos, emissão de recibos profissionais e acompanhamento financeiro de jobs.

## 🚀 Funcionalidades Principais
* **Painel Geral:** Visualização instantânea do total em caixa e valores a receber.
* **Emissão de Orçamentos:** Geração automática de PDFs profissionais com a identidade visual da SaarteSvm.
* **Gestão de Projetos:** Controle individual de status de produção e pagamentos (Entrada/Final/Integral).
* **Recibos Rápidos:** Emissão de comprovantes de pagamento em um clique.
* **Configuração Dinâmica:** Ajuste de contatos, endereço e informações da empresa em tempo real.

## 🛠️ Tecnologias Utilizadas
* **Python:** Linguagem principal do sistema.
* **Streamlit:** Interface web moderna e responsiva.
* **SQLite:** Banco de dados local para armazenamento seguro.
* **FPDF:** Motor de geração de documentos PDF.

## 📁 Estrutura do Repositório
* `main.py`: Código fonte principal do sistema.
* `requisitos.txt`: Lista de dependências para instalação.
* `saartesvm_data.db`: Banco de dados (gerado automaticamente na primeira execução).
